> itoa --- convert int to ascii?

## (1) MY CODES

~~~c
#include "libft.h"

int	ft_isascii(int c)
{
	return (c >= '0' && c <= '127');
}

int	ft_isprint(int c)
{
	return (c >= 32 && c <= 126);
}
~~~

- 정수를 매개 변수로 받아 그것이 아스키 코드에 속하는지, 출력 가능한 문자인지 판별하는 간단한 함수.
